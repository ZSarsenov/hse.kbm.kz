
	---------------Путеводитель SDK НУЦ РК---------------

	Данный комплект разработчика предназначен для информационных систем,
	внедряющих ЭЦП НУЦ РК в свою систему. 
	В SDK представлены ряд библиотек, примеров к ним и тестовые сертификаты. 
	
	Библиотеки можем разделить на 2 категории: для Front и Back. 
		- На сегодняшний день для Front можем предложить только одно решение - это NCALayer. 
		Пример использования основных методом модуля размещен на https://github.com/pkigovkz/NCALayerAnotherSampleBundle
		- Все остальные библиотеки, представленные в SDK предназначены для Back системы. 
	
	Состав SDK:
		1) Java (provider, utils, xmldsig) - каталог, содержащий библиотеки и примеры на Java для 
			Windows и Linux. Только для Back.  Описание - https://github.com/pkigovkz/sdkinfo/wiki/
		1.1) utils - каталог, который содержит вспомогательную библиотеку knca_provider_util и соответствующий javadoc. Данная библиотека
			содержит готовые методы для облегчения таких задач как проверка сертификата, создание подписи
			различных форматов, валидация и т.д. и т.п.
		
		2) COM_Windows - каталог, содержаший COM-библиотеки и примеры (C#, Go, VB, Python, Delphi, php) для Windows. 
			Только для Back. Описание методов библиотеки:   COM_Windows\docs\docs\Описание KalkanCryptCOM.docx
		
		3) PHP_Linux - каталог, содержаший php-библиотеки и пример (php) для Linux. Только для Back.
		
		4.1) С(Linux/C) - каталог, содержащий библиотеки *.so на С для Linux. Описание методов 
			библиотеки:   C\Linux\C\docs\Описание KalkanCrypt.docx. Только для Back.
		4.2) С(Linux/KalkanCrypt-OpenSSL) - каталог, содержащий основные библиотеки для обеспечения реализаций 
			многочисленных криптографических примитивов - иначе говоря "движок" криптографии для Linux.  Только для Back.
		4.3) С(Windows/KalkanCrypt_С) - каталог, содержащий библиотеки *.dll на С для Windows. Описание методов 
			библиотеки:   C\Linux\C\docs\Описание KalkanCrypt.docx. Только для Back.
		4.4) С(Windows/KalkanCrypt-OpenSSL) - каталог, содержащий основные библиотеки для обеспечения реализаций
			многочисленных криптографических примитивов - иначе говоря "движок" криптографии для Windows.  Только для Back.
		
		5) Keys and Certs - каталог, содержащий тестовые регистрационные свидетельства, CRL, корневые сертификаты и др.
