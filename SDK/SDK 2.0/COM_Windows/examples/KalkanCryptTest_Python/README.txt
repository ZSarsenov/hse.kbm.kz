Для реализации программы на языке PYTHON была создана форма (KalkanForm.ui) с помощью программы Designer, затем ее конвертировали в формат PYTHON: (KalkanForm.py).  
Для использования форм необходимо установить python-библиотеку PYQT5 (Устанавливать через командную строку: pip install pyqt5) 
 
Для использования KalkanCryptCOM-библиотеки необходимо установить дополнительную pythonбиблиотеку win32com.client (ее также устанавливать через командную строку: pip install pypiwin32)

Данный пример реализован и работает только под управлением ОС Windows
