----Инструкция----

В архивах test2022.zip и production2022.zip хранятся соответственно тестовые корневые сертификаты и боевые корневые сертификаты 

1) Разахивируйте архив.
2) Запустите из командной строки от имени администратора: если ubuntu: файл instal_*.sh или если CentOS и Fedora: instal_*centos_fedora.sh
